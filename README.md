## Prerequisites

- Node.js (LTS recommended)

## Run locally

```bash
cd "session 03 - slides (app)"
npm install
npm run dev
```

Vite will print a local URL (typically `http://localhost:5173`).

## Build / preview

```bash
cd "session 03 - slides (app)"
npm run build
npm run preview
```

## Where the slides live

- Slide content: `session 03 - slides (app)/src/slides.ts`
- Images: `session 03 - slides (app)/public/images/`
